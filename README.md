# sgpa-calculator

A clean and lightweight web application for calculating SGPA based on subject credits, internal marks, external marks, and grade points.  
This tool allows users to dynamically add subjects, enter marks, compute grade points, and generate the final SGPA using the standard credit-weighted formula.

## Features
- Add subjects with customizable credit values  
- Auto-calculation of totals, grade points, and credit contributions  
- Delete subjects dynamically  
- Save and load data using browser storage  
- Fully client-side (works on GitHub Pages)

## Tech Stack
- **HTML**
- **CSS**
- **JavaScript**
